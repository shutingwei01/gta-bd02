
# GTA1000 Series 语音板用户指南

## 概述

语音板是一款专为端侧语音交互设计的核心组件。该板卡合封了 512MB LPDDR3 颗粒的 GTA 主控芯片，支持丰富的端侧语音功能，无需二次开发即可适配各类前装或后装设备。产品体积小巧、接口灵活多变，满足不同场景下的集成需求。

**主要功能特性：**
* **端侧智能交互：** 支持离线/在线语音识别（ASR）、语音合成播报（TTS）、离线语音控制及全双工持续语音对话功能。
* **灵活架构设计：** 必须搭配独立麦克风板使用（通过 A2B/ASN 协议串接），麦克风板的规格与线束长度可根据设备形态灵活定制。
* **高安全性：** 板载专用加密芯片，实现本地模型与固件的安全保护。

## 子系统设计与技术说明

### 供电系统

语音板支持两种 **12V 3A** 供电接入方式，用户可根据设备结构任选其一：
* **方案A：** 使用工业级螺丝式端子（接口1）连接直流电源。
* **方案B：** 使用标准 Type-C 接口（接口3）连接电源适配器。

:::{warning}
**重要警告：**
1. 本板卡电源仅为语音板本体及其串接的麦克风板供电。
2. **扬声器必须独立供电**。如果您外接了功放扬声器，请务必为扬声器提供单独的适配电源与开关控制。语音板无法为扬声器直接供电。
:::

### 存储系统 (双模设计)

根据不同场景需求，主板提供两种 SD 存储方式（出厂时二选一配置，标配容量512MB）：
* **插卡式 (接口11)：** 便于日常维护和插拔，适合有频繁升级固件/更换模型需求且工作环境稳定的固定设备。
* **贴片式 (芯片14)：** SMD 焊接固定，抗震动性能优异，稳定性极佳。适合车载或震动环境下的工业设备（注意：升级文件相对不便）。

### 数据与通信系统

语音板采用 UART串口与客户主机设备通信，波特率等参数均开放配置。提供两种物理连接方式：
* **USB直连：** 通过接口10 (USB-A) 连接。主板内置 TTL-USB 转换芯片，即插即用，适用于无原生串口的电脑或安卓主板。
* **TTL直连：** 通过接口12连接。支持直接使用 TTL 信号通信，用户也可在此接口外接 RS232 或 RS485 转接板，满足工业总线需求。

## 4. 硬件安装与快速上手

### 4.1. 开箱与准备
我们将根据您的采购需求提供全套配件，包含：`语音板`、`麦克风板`、`配套线束`、`电源适配器`、`扬声器`、`USB线/通信模块` 及 `SD卡`（若选插卡版）。

### 4.2. 硬件固定
1. **语音板固定：** 在设备内部寻找平整位置，使用螺丝将语音板四角的固定孔拧紧。
2. **麦克风板固定：** 根据设备外观开孔位置固定麦克风板。
3. **连接两者：** 使用配套线束将麦克风板接入语音板的 **[接口7]**。注意理顺线材走向，避免拉扯。

### 4.3. 连接外设与通信端
1. **插入存储卡：** 若选用插卡式，请将 SD 卡插入 **[接口11]**。
2. **连接扬声器：** 将外部功放扬声器的 3.5mm 音频线插入语音板的 **[接口6]**。扬声器请安装在合适位置。

:::{note}
为保证拾音和识别率，扬声器喇叭应尽可能远离麦克风的收音孔位置，以减少声学回声干扰。
:::

3. **连接控制主机：** 
   * **USB 方案：** 使用 USB 数据线连接主机的 USB 接口与语音板的 **[接口10]**。
   * **串口 方案：** 将主机的 TX/RX 等线缆接到语音板的 **[接口12]**（或通过 RS232/485 模块转接）。

### 4.4. 通电与测试
1. 确认所有线缆连接稳固。
2. 将扬声器连接独立电源并开启（若有）。
3. 将 12V 供电接入语音板的 **[接口1]** 或 **[接口3]**。
4. 拨动 **[接口2]** 的主电源开关至 `ON` 状态。
5. 系统上电后即开始工作，您可下发指令或进行语音唤醒测试。如需重新初始化，可短按 **[接口9]** 复位键。
